Problem 1 — Coding (Medium)

Longest Subarray with At Most K Distinct Characters

Given a string s and integer k, return the length of the longest substring containing at most k distinct characters.

Example: s='eceba', k=2 → 3 (substring 'ece'). s='aa', k=1 → 2.

State before coding: pattern name, time complexity, space complexity.

Pattern: _____________ Time: _____________ Space: _____________

Reference:

Variable sliding window. Use a HashMap tracking character frequencies in the window. Expand right, shrink left when distinct count > k. Time O(n), Space O(k). Key: shrink loop runs while map.size() > k, removing characters whose frequency drops to 0.

Problem 2 — Coding (Hard)

Word Ladder — Shortest Transformation Sequence

Given beginWord, endWord, and a dictionary wordList, find the length of the shortest transformation sequence from beginWord to endWord, changing exactly one letter at a time. Each intermediate word must be in wordList. Return 0 if no such sequence exists.

Example: beginWord='hit', endWord='cog', wordList=['hot','dot','dog','lot','log','cog'] → 5.

State: why BFS and not DFS? How do you generate neighbours efficiently?

Pattern: _____________ Time: _____________ Space: _____________

Reference:

BFS — shortest path in unweighted graph. Convert wordList to HashSet for O(1) lookup. For each word in queue, generate neighbours by changing each character to a-z and checking if in set. BFS level = transformation steps. Time O(M^2 * N) where M=word length, N=wordList size. Bidirectional BFS reduces this significantly.

System Design — Rate Limiter

Design a rate limiter service used by thousands of microservices.

Requirements: limit API calls per user, per service, and per endpoint. Support multiple algorithms (token bucket, sliding window). 100k requests/second peak. The limiter itself must be highly available — it cannot be the bottleneck. Clients should get clear 429 responses with Retry-After. Design the API, the data store, the algorithm choice, and how you make the limiter itself resilient.
