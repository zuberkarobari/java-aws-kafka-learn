Core Java — Rapid Recall

Question

Answer — cover and say from memory

HashMap bucket index formula?

(n-1) & hash(key) where n is always a power of 2. hash() = h ^ (h>>>16).

Treeify threshold?

8 nodes in a bucket → Red-Black tree. Untreeify at 6.

ConcurrentHashMap read — locking?

No lock. Reads use volatile. Writes to empty bucket use CAS. Writes to occupied bucket sync on first node.

volatile guarantees?

Visibility (always read from main memory) and ordering (writes happen-before reads). NOT atomicity.

happens-before for volatile?

Writing to volatile field happens-before every subsequent read of that same field.

CompletableFuture.join() vs get()?

Both block. get() throws checked exceptions. join() throws unchecked CompletionException.

ReentrantLock over synchronized — when?

When you need tryLock, lockInterruptibly, fair ordering, or multiple condition variables.

Java memory areas: what goes where?

Heap: objects. Metaspace: class metadata. Stack: frame per method call. Native: JNI, direct buffers.

G1GC target pause flag?

XX:MaxGCPauseMillis=200. This is a target, not a guarantee.

ZGC pause characteristic?

Sub-millisecond pauses regardless of heap size. Slightly lower throughput than G1GC.

Spring Boot — Rapid Recall

Question

Answer — cover and say from memory

@SpringBootApplication = ?

@SpringBootConfiguration + @EnableAutoConfiguration + @ComponentScan

How auto-config loads?

AutoConfigurationImportSelector reads ImportCandidates, evaluates @Conditional, registers passing beans.

@ConditionalOnMissingBean purpose?

Only create the bean if user has not already defined one of the same type. This is the override mechanism.

Bean lifecycle step 4?

BPP.postProcessBeforeInitialization runs for all registered BPPs.

AOP proxy type for classes without interface?

CGLIB — subclasses the concrete class. Cannot proxy final classes or final methods.

4 @Transactional traps?

Self-invocation, private/final method ignored, checked exceptions don't rollback, REQUIRES_NEW inside test @Transactional.

REQUIRED vs REQUIRES_NEW?

@Async void exception handling?

Silently swallowed unless AsyncUncaughtExceptionHandler is configured.

SimpleAsyncTaskExecutor problem?

Creates new thread per @Async call — no pooling, no bounds. OOM under load.

Actuator: which endpoints to restrict?

/env, /heapdump, /beans, /loggers. Only expose /health, /metrics, /info publicly.

Testing — Rapid Recall

Question

Answer — cover and say from memory

@WebMvcTest excludes what?

@Service, @Repository, JPA, all non-web beans. Dependencies must be @MockBean.

@MockBean vs @Mock?

@MockBean registers the mock in the Spring ApplicationContext. @Mock is pure Mockito with no Spring context.

Why Testcontainers over H2?

H2 is not PostgreSQL — no JSONB, no pg_trgm, different query planner. Tests can pass on H2 and fail on real PostgreSQL.

Object Mother pattern?
