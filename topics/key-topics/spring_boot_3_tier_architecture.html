1. Controller
Handles HTTP routing, extracts JSON/params, maps to DTOs, and returns HTTP status codes. Never calculates business rules.
2. Service
Independent of HTTP. Executes core domain logic, calculates data, and manages @Transactional boundaries.
3. Repository
Strictly translates Java operations into SQL/NoSQL queries. Abstracted via Spring Data JPA.
