The 400 vs 422 vs 409 Rule
CodeMeaningExample Scenario
400 Bad Request
MALFORMED		 Syntax error, missing curly brace, totally invalid JSON format.
422 Unprocessable
VALIDATION		 Valid JSON, but business rule fails (e.g. age < 0).
409 Conflict
STATE		 Request is perfectly valid, but DB state rejects it (e.g. Email exists).

Offset vs Cursor
OFFSET (O(n)):
LIMIT 100 OFFSET 500000
DB scans and discards 500k rows. Extremely slow.

CURSOR (O(1)):
WHERE (created_at, id) < (?, ?)
Uses index to jump directly to exact row. ID resolves duplicate timestamps.

The Truth About HATEOAS
"Academics love HATEOAS (Hypermedia links in JSON). But in reality, unless you are building a highly generic, dynamically evolving platform, the overhead of managing dynamic links severely outweighs the benefits for standard mobile/web clients."
