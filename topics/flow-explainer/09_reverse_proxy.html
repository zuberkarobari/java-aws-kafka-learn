9. Reverse Proxy (NGINX)

Why don't we expose Spring Boot's Tomcat directly to the internet? Enter the Reverse Proxy: handling SSL, compression, and static files with C-level speed.

1. Definition

Simple English

If you call a celebrity's phone, their publicist answers. The publicist filters out the junk, handles the paparazzi, and only forwards the important calls to the celebrity's private number. The Reverse Proxy is the publicist.

Technical

A server that sits in front of one or more web servers, intercepting requests from clients. It provides caching, load balancing, SSL termination, and protection against attacks, forwarding the purified request to the backend server.

Purpose

To protect the application server (like Tomcat in Spring Boot) from direct internet exposure. It handles slow client connections and heavy CPU tasks like TLS encryption so the Java app can focus on business logic.

2. Internal Working: SSL Termination & Forwarding

NGINX is often installed on the exact same EC2 instance as the Spring Boot app, or just in front of a cluster of them. It intercepts Port 443, does the heavy lifting, and forwards to Port 8080.

3. Interview Questions

Answer: SSL Termination is the process of decrypting encrypted HTTPS traffic before passing it along to the web server.

The Reverse Proxy holds the SSL Certificate and performs the CPU-intensive TLS Handshake with the client. It decrypts the payload, and then forwards a plain HTTP request over the internal, secure network to the Spring Boot application. This significantly reduces the CPU load on the Java application.

Answer:

A Forward Proxy sits in front of the Client. It protects the client by masking their IP address before forwarding their request to the internet (like a corporate proxy or a VPN). The server doesn't know who the client is.

A Reverse Proxy sits in front of the Server. It protects the server by intercepting requests from the internet before they reach the backend. The client doesn't know what backend server actually processed the request.
